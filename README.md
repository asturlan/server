### TTM-server
Timetable maker - server side

##Installation and setup
`npm install`

#Setting up DB variables
Navigate to -env file and change DB_* variables to math your DB environment

##Development
`npm run start`

##Deployment
```
npm run build
npm run build-start
```

##Built with
- Express
- Postgres
- Sequelize
